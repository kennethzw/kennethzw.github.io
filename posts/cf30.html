Linux终端命令格式

command [-options] [parameter]

  • command 命令名,相应功能的英⽂单词或单词的缩写
  • [-options] 选项,可⽤来对命令 进⾏控制，也可以省略
  • parameter 传给命令的参数：可以是零个 ⼀个或多个

常用Linux命令的基本使用

命令说明
ls查看当前文件夹下的内容
pwd查看当前所在文件夹
cd [目录名]切换文件夹
touch [文件名]如果文件不存在，新建文件
mkdir [目录名]创建目录
rm [文件名]删除指定文件名
clear清屏

查阅命令帮助信息

命令说明
command --help显示command命令的帮助信息
man command查阅command的使用手册

小技巧

命令说明
Ctrl + Shift + =放大终端窗口的字体显示
Ctrl + -缩小端窗口的字体显示
Tab自动补全
上/下选择曾经使用命令
Ctrl + c退出选择
